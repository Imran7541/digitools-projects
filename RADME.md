# 🚀 DigiTools – Digital Tools Buying Platform

## 📌 Project Name

**DigiTools – Smart Digital Tools Marketplace**

---

## 📖 Description

DigiTools is a modern and responsive web application where users can explore and purchase various digital tools such as resume builders, AI tools, and productivity software.
It provides a smooth user experience with dynamic product listing, cart management, and interactive UI components.

---

## ⚙️ Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS & DaisyUI
* 🧠 JavaScript (ES6+)
* 🔔 React Toastify
* 📦 JSON (for product data)

---

## ✨ Features

### 🛒 1. Smart Cart System

* Add products to cart instantly
* Remove items easily
* Shows total price dynamically
* Checkout clears the cart

### 🔄 2. Dynamic Toggle System

* Switch between **Products** and **Cart** view
* Default view shows products
* Clean and interactive UI

### 📊 3. Modern UI & Responsive Design

* Fully responsive for mobile, tablet, and desktop
* Clean layout based on Figma design
* Smooth user experience with beautiful components
